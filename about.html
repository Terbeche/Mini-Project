ROBOTS ARENA
CAUSE YOU DESERVE THE BEST GAMING EXPERIENCE

Is a video game that is inspired from { Quake, Unreal tournament and Karlson } that takes place in a fictional world that overrun by robots and you (the player) are the last line of defense You need to eliminate the enemies in order to survive and be the winner.

The objective of this alpha build of the game is to gather information on the player base and to test the network features of the game alongside publicity for the actual release and a potential startup.
